# Checklist Pre-Deploy a Producción

## Funcionalidad
- [ ] Todos los cambios funcionan en staging
- [ ] Probado en móvil y desktop
- [ ] Probado en Chrome, Firefox, Safari
- [ ] Formularios funcionan correctamente
- [ ] Links internos y externos funcionan

## Rendimiento
- [ ] Imágenes optimizadas
- [ ] Sin errores en consola del navegador
- [ ] Tiempo de carga aceptable

## SEO/Contenido
- [ ] Meta titles y descriptions correctos
- [ ] Sin contenido de prueba visible
- [ ] URLs limpias

## Código
- [ ] Sin console.log o código de debug
- [ ] Sin credenciales hardcodeadas
- [ ] .gitignore actualizado

## Backup
- [ ] Backup de producción realizado (SiteGround lo hace automático)
- [ ] Sé cómo hacer rollback si algo falla

## Aprobación
- [ ] Cliente/stakeholder ha aprobado los cambios en staging
- [ ] Fecha y hora de deploy acordada
