# Prompt para Figma AI: Página Consultoría de Tecnología

## Instrucciones para Figma

Copia y pega este prompt en Figma AI (Make Designs) para generar el diseño de la página.

## PROMPT PRINCIPAL

```
Diseña una página de servicio de consultoría de tecnología para Proportione, una firma de consultoría estratégica española. La página debe tener 9 secciones con un estilo minimalista sofisticado, profesional y corporativo.

## DESIGN SYSTEM OBLIGATORIO

### Paleta de Colores
- Granate principal: #5F322F (overlays, bordes, acentos)
- Verde oliva: #6E8157 (botones primarios, iconos, números destacados)
- Verde oscuro: #3B431C (hover de botones verdes)
- Crema: #F5F0E6 (fondos alternos, texto sobre granate)
- Blanco: #FFFFFF (fondos de cards, secciones alternas)
- Texto: #333333 (párrafos, títulos sobre fondo claro)

### Tipografía
- Títulos (H1, H2, H3): Oswald, pesos 600-700
- Cuerpo y botones: Raleway, pesos 400-500
- Escala: H1=56px, H2=42px, H3=28px, Body=18px, Small=14px

### Espaciado
- Padding secciones: 100px vertical (60px en móvil)
- Max-width contenido: 1140px (900px para texto centrado)
- Border radius: 8px
- Sombras cards: 0 4px 20px rgba(0,0,0,0.08)

## NOTAS DE DISEÑO

1. **Animaciones sugeridas**: Fade In Up para títulos, Fade In Left/Right para columnas
2. **Hover en cards**: translateY(-4px) + sombra más intensa
3. **Hover en botones**:
   - Primario: fondo #3B431C (verde más oscuro)
   - Secundario: fondo granate, texto crema
4. **Responsive**:
   - Tablet: 2 columnas en grids, H1 48px
   - Móvil: 1 columna, H1 36px, botones full-width apilados
5. **Sin línea decorativa bajo H2** en secciones con fondo oscuro (Hero, Summary)
```
