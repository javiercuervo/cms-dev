## PATRONES CLAVE IDENTIFICADOS (SÍNTESIS TRANSVERSAL)

### Dimensión 1: Filosofía Subyacente

**Principio Unificador: "Profesionalismo Inteligente"**

```
Profesionalismo Corporativo + Inteligencia Sofisticada = Autoridad sin Arrogancia

Componentes:
├─ Austeridad visual (minimalismo) = seriedad
├─ Precisión conceptual (lenguaje) = expertise
├─ Claridad extrema (tipografía, contraste) = respeto por el lector
├─ Espacios generosos (blank space) = confianza
└─ Autoridad demostrada (casos, métricas) = comprobación de capacidad
```

### Dimensión 2: Restricción Creativa

**Patrón Observable: Máxima Impacto con Mínimos Elementos**

| Aspecto | Restricción | Propósito |
|---------|-------------|-----------|
| **Colores** | 1 color primario + neutros | Enfoque, no distracción |
| **Tipografía** | 1 familia sans-serif | Coherencia, legibilidad |
| **Espacios** | Generosos, no comprimidos | Respiración visual |
| **Animaciones** | Sutiles, funcionales | Feedback sin distracción |
| **Imágenes** | Pocas, cuando usadas: impactantes | Legitimidad visual |
| **Decoración** | Ninguna | Contenido es diseño |

### Dimensión 3: Audiencia Objetivo

**Perfil Implícito: CEO/Tomador de Decisión Estratégica**

```
Características de Audiencia:
├─ Inteligencia: Alta (no necesita simplificación patronizante)
├─ Urgencia: Pressurizado (eficiencia máxima en lectura)
├─ Presupuesto: Importante (solicitud de presupuesto disponible)
├─ Tiempo: Escaso (menos de 5 minutos por página típicamente)
└─ Expectativa: Experto, respetado como par

Implicaciones de Diseño:
├─ Lenguaje: Sofisticado, no didáctico
├─ Visual: Profesional, no lúdico
├─ Navegación: Eficiente, no exploratorio
├─ Contenido: Conciso con profundidad disponible
└─ CTA: Claro, no agresivo
```

## MATRIZ DE PATRONES REUTILIZABLES

### Para Componentes de Contenido

```
PROPOSICIÓN (Texto)
├─ Tamaño: 3-8 palabras
├─ Tono: Aspiracional, nominal
├─ Ejemplo: "Unlocking the Potential..."
└─ Propósito: Anclaje conceptual

DESCRIPCIÓN DE CAPACIDAD (Texto)
├─ Estructura: Qué | Cómo | Por qué
├─ Largo: 150-250 palabras
├─ Tono: Capaz, colaborativo
└─ Propósito: Explicar enfoque sin jargon

CASO DE ÉXITO (Texto + Métricas)
├─ Estructura: Desafío | Enfoque | Resultado
├─ Largo: 300-400 palabras
├─ Métrica: Siempre presente y específica
└─ Propósito: Proof point cuantificado

CTA (Texto)
├─ Verbo: Explore, Discover, Learn, Connect
├─ Objeto: Claridad total
├─ Ejemplo: "Discover how to unlock value"
└─ Propósito: Invitación clara, no agresiva
```

### Para Componentes Visuales

```
HERO SECTION
├─ Altura: 500-700px
├─ Tipografía: 80-120px (H1), 18-24px (subtítulo)
├─ Colores: 1 fondo + texto alto contraste
└─ Propósito: Impacto visual, anclaje

GRID DE SERVICIOS
├─ Columnas: 3 desktop, 2 tablet, 1 mobile
├─ Gutter: 24-32px consistente
├─ Card altura: 250-300px uniforme
└─ Propósito: Escaneo visual rápido

BODY COPY (Sección de Texto)
├─ Tipografía: 16px, 400 weight
├─ Line-height: 1.6-1.8
├─ Max-width: 800px
├─ Padding: 40-60px top/bottom
└─ Propósito: Lectura confortable prolongada

ESPACIADO VERTICAL
├─ Entre secciones: 80-120px
├─ Entre elementos: 24-40px
├─ Padding interno: 24-32px
└─ Principio: Proporción 1:1.5:2 típica
```

## CHECKLIST DE COHERENCIA INTEGRADA

### Textual

- [ ] ¿Hay proposición clara de valor (máximo 8 palabras)?
- [ ] ¿Proposición es aspiracional pero demostrada?
- [ ] ¿Verbos son transitivos con objetos claros?
- [ ] ¿Se evita primera persona (I/me/my)?
- [ ] ¿Color primario usado máximo 5-10% de página?
- [ ] ¿Todo párrafo tiene máximo 4 líneas (80 palabras)?
- [ ] ¿Existe evidencia (métrica/caso) para afirmación de capacidad?
- [ ] ¿Links internos (mínimo 2-3) a contenido relacionado?

### Visual

- [ ] ¿Contraste texto mínimo 4.5:1 (WCAG AA)?
- [ ] ¿Body copy 14-16px mínimo?
- [ ] ¿Espacios en blanco entre secciones ≥80px?
- [ ] ¿Grid de 12 columnas respetado?
- [ ] ¿Máximo 2-3 colores saturados por sección?
- [ ] ¿Hover states claros en botones?
- [ ] ¿Focus rings visibles en elementos interactivos?
- [ ] ¿Transiciones suaves (200-400ms)?
- [ ] ¿Mobile responsive (320px, 768px, 1024px+)?

### Integración

- [ ] ¿Tono textual coincide con lenguaje visual (profesional, no casual)?
- [ ] ¿Jerarquía visual (tamaño, peso) coincide con jerarquía conceptual (importancia)?
- [ ] ¿Espaciado visual permite lectura sin fatiga?
- [ ] ¿Colores refuerzan propuestas (primario en CTAs, gris en secundarios)?
- [ ] ¿Tipografía comunica mismo nivel de sofisticación que contenido?

## CÓMO USAR ESTOS DOCUMENTOS PARA ENTRENAR IA

### Fase 1: Lectura e Internamiento
1. Leer `guia-comunicacion-consultoria.md` completamente
2. Leer `guia-diseno-visual-consultoria.md` completamente
3. Crear mental model de interacción entre ambas dimensiones

### Fase 2: Aplicación en Prompts
**Formato de Prompt para Generación de Contenido:**

```
Genera contenido para [tipo de página] respetando:

TEXTUAL:
- Tono: [aspiracional/diagnóstico/narrativo]
- Proposición: [máximo 8 palabras, aspiracional]
- Estructura: [qué/cómo/por qué, con evidencia]
- Largo: [300-500 palabras para body, etc.]

VISUAL:
- Altura sección: [500-700px para hero, etc.]
- Tipografía: [36px semibold para H2, etc.]
- Espaciado: [80px entre secciones]
- Colores: [blanco fondo, gris oscuro texto, azul CTA]

INTEGRACIÓN:
- ¿El contenido y visual comunican igual nivel de sofisticación?
- ¿Existe coherencia entre tono textual y visual?
- ¿Hay evidencia visual (métricas, casos, imágenes) para afirmaciones textuales?
```

### Fase 3: Iteración y Refinamiento
1. Generar componente (contenido o visual)
2. Verificar contra checklist correspondiente
3. Ajustar según feedback
4. Validar integración entre textual y visual

## DIFERENCIADORES CLAVE VERSUS COMPETENCIA DÉBIL

### Signos de Consultoría de Baja Autoridad (Antipatrones)

| Elemento | Débil | Fuerte |
|----------|-------|--------|
| **Proposición** | "Best-in-class solutions" | "Unlocking competitive advantage" |
| **Verbo** | "Leverage", "Synergize" | "Unlock", "Transform", "Build" |
| **Color** | Múltiples colores saturados | Monocromo + 1 acento |
| **Espaciado** | Comprimido, denso | Generoso, respiración |
| **Evidencia** | Promises sin validación | Casos + métricas específicas |
| **Tipografía** | Serif ornamental | Sans-serif limpio |
| **CTA** | "Click here!", "Limited time!" | "Explore", "Discover", "Learn" |
| **Imágenes** | Stock photos obvias | Fotografía corporativa auténtica |
| **Contraste** | Gris sobre gris | Negro sobre blanco |
| **Navegación** | Múltiples CTAs competidores | Clear primary + secundarias |

## SÍNTESIS FINAL

### Patrón de Éxito Comprobado

La combinación de:

1. **Lenguaje preciso y aspiracional** (comunicación textual)
2. **Diseño minimalista y legible** (comunicación visual)
3. **Evidencia cuantificada** (credibilidad)
4. **Coherencia absoluta** (ambas dimensiones refuerzan la misma propuesta)

...genera percepción de:
- Inteligencia institucional
- Confianza y seriedad
- Sofisticación sin arrogancia
- Accesibilidad a pesar de complejidad

### Principio Unificador

> "Máxima claridad, máxima sofisticación, máximo impacto con mínimos elementos."

Esto es lo que distingue consultoría de autoridad (3 mencionadas) de competencia débil.
