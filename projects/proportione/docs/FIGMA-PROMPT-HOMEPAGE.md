# Prompt para Figma - Homepage Proportione

## Resumen

Prompt completo y detallado para que Figma AI (o un diseñador) diseñe la homepage de Proportione, una consultora de transformación digital.

### Diferencias clave entre web vieja y nueva:
- **Headline**: De "Estrategia, tecnología y personas" → "Digitalización sin complicaciones" (más orientado a beneficio)
- **Colores**: De magenta/rosa (#f00069) → Granate (#5F322F) (más sofisticado)
- **Metodología**: Nueva sección 20-60-20 prominente
- **Tono**: Más directo, orientado a conversión

### El prompt incluye:
- ✅ Colores exactos con HEX
- ✅ Tipografías específicas
- ✅ Textos literales de cada sección
- ✅ Estructura visual con dimensiones
- ✅ Estados de hover y interacciones
- ✅ Principios de diseño
- ✅ Emociones objetivo
- ✅ Lo que evitar vs preferir
